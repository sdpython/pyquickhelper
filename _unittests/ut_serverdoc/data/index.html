Description¶

This extension gathers three functionalities:
  • a logging function: fLOG
  • a function to synchronize two folders: pyquickhelper.synchronize_folder
  • a function to generate a copy of a module, converting doxygen documentation in rst format: generate_help_sphinx (see also prepare_file_for_sphinx_help_generation)

The module is available on pypi/pyquickhelper and on GitHub/pyquickhelper.

Functionalities¶

  • help generation
  • folder synchronization
  • logging
  • import a flat file into a SQLite database
  • help running unit tests
  • functions to convert a pandas DataFrame into a HTML table or a RST table
